# Calculus III Tutoring Materials — 2026.1 (UESB)

This repository contains slides and supporting materials for Calculus III tutoring sessions at the Universidade Estadual do Sudoeste da Bahia (UESB), during the 2026.1 academic semester.

The materials follow the course syllabus and are organized to support topics such as series, multivariable calculus, Fourier analysis, and vector calculus, with applications to computational problems, mathematical modeling, and algorithm analysis.

---

## Repository Structure

The repository is organized into three units:

```text
.
├── unit-1/
├── unit-2/
└── unit-3/
```


Each directory contains slides and may also include additional resources such as notes, exercises, or code implementations.

---

## Topics Covered

### Unit 1
- Numerical and functional series: convergence and divergence  
- Power series and function approximation  
- Taylor formula and applications  
- Taylor and Maclaurin series  

### Unit 2
- Function approximation in numerical algorithms  
- Computational error estimation  
- Partial derivatives and gradient  
- Critical points and function optimization  
- Constrained optimization (Lagrange multipliers)  

### Unit 3
- Fourier series and Discrete Fourier Transform (DFT)  
- Applications in signal processing, digital images, and data compression  
- Fast Fourier Transform (FFT)  
- Vector fields  
- Gradient, divergence, and curl  
- Green’s, Gauss’s, and Stokes’ theorems  
- Line and surface integrals
